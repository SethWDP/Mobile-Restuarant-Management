1. Project Title
   Restaurant Management System

This project is a Restaurant Management System that allows administrators to manage food menu items through a mobile application. The system supports adding, updating, deleting food items and uploading food images using a RESTful API.

2. Features

List the main functions of system.
View food menu
Add new food items
Update food information
Delete food items
Upload food images
API communication between mobile app and server

3. Technologies Used

Flutter – Mobile application UI
Node.js + Express.js – Backend REST API
MySQL – Database storage
XAMPP – Local server
Postman – API testing

4. System Architecture
   User/Admin
   |
   Flutter Mobile App
   |
   Node.js + Express API
   |
   MySQL Database

5. Installation / Setup

6. Clone the repository
   git clone https://github.com/SethWDP/Mobile-Restuarant-Management.git
7. Install backend dependencies
   cd restuarant_api
   npm install
8. Start the server
   node server.js
9. Run Flutter application
   start emulator : flutter emulators --launch your emulators name
   flutter run
